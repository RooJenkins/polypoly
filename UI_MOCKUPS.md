# Polystocks UI Mockups - 4 Design Concepts

## Current Data to Display
- **Agent Performance**: Name, Account Value, ROI%, Total P&L
- **Trading Activity**: Recent trades, open positions
- **Performance Metrics**: Win rate, Sharpe ratio, max drawdown, biggest win/loss
- **Time Series**: Account value history chart
- **Real-time**: Stock ticker, live updates

**Design Principles**:
- Massive typography for key numbers
- Apple-style spacing (80-100px between major sections)
- Clean sans-serif throughout (SF Pro, Inter, or System UI)
- No borders - pure whitespace separation
- Subtle cards with minimal shadow
- Information organized in scannable chunks
- Monospace for numbers only

- Background: Pure white (#FFFFFF)
- Text: Near black (#1A1A1A)
- Secondary text: Medium gray (#6B7280)
- Subtle borders: Very light gray (#F3F4F6)
- Positive: Muted green (#059669)
- Negative: Muted red (#DC2626)
- Accents: Pure black for emphasis
